# react-forcefield

Prevent unintentional interactions until click or touch.

<Head>
  <title>react-forcefield</title>
</Head>

```.jsx
<Forcefield>
    <iframe
      src="https://basic-map.coston.io"
      style={{ border: '1px #eee solid' }}
      name="A basic map"
      scrolling="no"
      height="800px"
      width="100%"
      allowFullScreen
    />
</Forcefield>
```
