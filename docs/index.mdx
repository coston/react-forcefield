# react-forcefield 

Prevent unintended interactions until click or touch.

## Demo 
Map scroll zoom is inactive until click.
```.jsx
<Forcefield>
    <iframe
      src="https://basic-map.coston.io"
      style={{ border: '1px #eee solid' }}
      height="800px"
      width="100%"
    />
</Forcefield>
```
